## About The Project


A fun simple trivia game where you as a user can register and answer some questions regarding a vareity of categories. You can also replay the game and try to beat your highscore. 

The project was mainly build with Vue.js framework and using TailwindCSS as the styling library.

<!-- USAGE EXAMPLES -->
## Usage

1. The game starts with that you as a user fill in a couple of parameters for your gameplay
  
       1.1 First your username
  
       1.2 Number of questions
  
       1.3 Difficulty
  
       1.4 Category
  
  
2. When all parameters have been filled in your press "Start game"
3. When all questions have been answered, you will get the results of your session, displaying all questions with your answer and the correct answer. You also get a score, each right answer collects you 10 points which adds up to a total score.
4. You can hit the "Replay" button which takes you back to the start screen where you do it all over again. Try to beat your highscore.
